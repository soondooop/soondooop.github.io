# 국제배송 부피무게 계산기

국제배송 시 실제 무게와 부피무게를 비교하여 운임 기준 무게를 계산하는 웹 애플리케이션입니다.

## 기능

- 가로, 세로, 높이 입력 (cm 단위)
- 실제 무게 입력 (kg)
- 제수 선택 (5000 또는 6000)
- 부피무게 자동 계산
- 실제 무게와 부피무게 중 큰 값을 운임 기준 무게로 표시

## 사용 방법

1. `index.html` 파일을 웹 브라우저에서 열기
2. 박스의 가로, 세로, 높이를 cm 단위로 입력
3. 실제 무게를 kg 단위로 입력
4. 제수를 선택 (일반적으로 5000 사용)
5. "계산하기" 버튼 클릭

## 계산 공식

```
부피무게(kg) = (가로 × 세로 × 높이) / 제수
운임 기준 무게 = max(실제 무게, 부피무게)
```

## 제수 설명

- **5000**: 일반적으로 DHL, FedEx 등 대부분의 국제택배사에서 사용
- **6000**: 일부 택배사에서 사용하는 제수

## 예시

- 가로: 50cm, 세로: 40cm, 높이: 30cm
- 실제 무게: 5kg
- 제수: 5000

**계산 결과:**
- 부피무게: (50 × 40 × 30) / 5000 = 12kg
- 실제 무게: 5kg
- **운임 기준 무게: 12kg** (부피무게가 더 크므로)
